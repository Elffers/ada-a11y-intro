Using VoiceOver to Read

To read content with VoiceOver (VO), we need to do a couple of things:

  1. Press Control + Option + Shift + Down Arrow to tell VO you want to get into the HTML of the page.
  2. Press Control + Option + Right Arrow to navigate to each element in order.
  3. Press Control + Option + Left Arrow to go back.

Fun fact: Control + Option is VO's shortcut set. All VO-specific shortcuts start with it!
